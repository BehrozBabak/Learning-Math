
ریاضی‌بازی - نسخهٔ v6 (موبایل-محور)
این بسته شامل یک PWA موبایل-فرست است که:
- اعداد و پاسخ‌ها همیشه صحیح (عددی صحیح) هستند؛ هیچ اعشاری وجود ندارد.
- محدوده‌های سطوح و توزیع‌های احتمال برای انتخاب اعداد و تعداد عملیات پیاده‌سازی شده.
- تصاویر بزرگ فقط در سطح مبتدی نمایش داده می‌شوند.
- صداها: فایل‌های کوتاه برای correct/wrong و یک لوپ پس‌زمینه قرار دارند.
- فونت B-kodak در بسته قرار ندارد؛ برای استفاده حتماً فایل فونت را در پوشه fonts/ قرار دهید و نام آن را نگه دارید.
